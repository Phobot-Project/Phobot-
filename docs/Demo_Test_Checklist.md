# Demo Test Checklist

## Before Demo
- [ ] Kobuki charged and powered on
- [ ] Samsung A17 charged (50%+)
- [ ] App installed on real phone (NOT emulator)
- [ ] USB cable / BT adapter / WiFi bridge ready
- [ ] Test area flat and clear

## USB Mode
- [ ] USB cable connected, permission allowed
- [ ] USB mode selected, Connect pressed
- [ ] Status green with VID/PID/Chip info
- [ ] STOP tested (robot stays still)
- [ ] FORWARD/STOP tested
- [ ] BACKWARD/STOP tested
- [ ] LEFT/STOP tested
- [ ] RIGHT/STOP tested
- [ ] Disconnect works

## Bluetooth Mode
- [ ] HC-05/HC-06 paired in Settings
- [ ] BT mode selected, device picked
- [ ] Connect pressed, status blue
- [ ] Movement commands work wirelessly
- [ ] Disconnect works

## WiFi Mode
- [ ] ESP powered on, phone on the same network
- [ ] WiFi mode selected, IP and port entered
- [ ] Connect pressed, status yellow
- [ ] Movement commands work wirelessly
- [ ] Disconnect works

## Voice (any mode)
- [ ] Mic permission granted
- [ ] "forward" recognized and executed
- [ ] "stop" recognized and executed

## Notes
- Phone: Samsung A17 (Android 16)
- Robot: Kobuki (Quanser/Yujin Robot)
- Tester: Atif, Sifat, Zed, Talha, Hafiz & Jackson.
